# MVP Drift

Минимальный проект на Phaser 3 с Matter.js для GitHub Pages.

## Требования

- Работает через CDN.
- Использует ES модули.
- Без сборщиков (Webpack/Vite не требуются).

## Управление

- **W / S** или **↑ / ↓** — Газ / Тормоз.
- **A / D** или **← / →** — Руль.
- **Space** — Ручной тормоз.
- **Shift / Ctrl** или **E / D** — Переключение передач (вверх / вниз).
- **R** — Сброс позиции.

## Основные возможности

- **Реалистичный дрифт**: RWD физика, расчет боковых сил, влияние ручника.
- **Двигатель и КПП**: RPM, крутящий момент, 5-ступенчатая коробка передач.
- **Продвинутая камера**: Плавное следование с опережением по скорости.
- **Трасса и круги**: Физические границы, система чекпоинтов и учет времени круга.
- **HUD**: Тахометр, спидометр, индикация времени круга и рекордов.
- **Сохранение данных**: Хранение лучшего времени и параметров физики в `localStorage`.
- **Визуальные эффекты**: Динамические следы резины (skidmarks) и ретро-шум.

## Структура проекта

- `index.html` — Точка входа.
- `src/main.js` — Инициализация игры.
- `src/track/Track.js` — Описание трассы, стен и чекпоинтов.
- `src/entities/Car.js` — Сущность машины (физика + визуал).
- `src/physics/CarPhysics.js` — Логика физики движения.
- `src/ui/HUD.js` — Игровой интерфейс.
- `src/storage/Storage.js` — Система сохранения данных.
- `src/scenes/Game.js` — Основная сцена игры и логика гонки.
